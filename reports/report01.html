ウェブサイトに関しての分析

概要

カラーコードが乗っているサイト

https://visualize60.ndc.co.jp/

考察

分析 背景が和紙ベースになっていて、「日本カラー」だということが伝わりやすい。 カラーコードが視覚的に分かるため、見やすい。 気になる色をクリックするとその色が背景色になるので、実際にどんな色味なのかパッとわかる。 SNSに繋がっている。 C、M、Y、Kの数値をグラフを用い、度合いを分かりやすくしている。 気になる色をクリックした際、色の明暗によって色が変わっていくアニメーションに違いがある。 他のページとの比較 実際にカラーコードを使うまで、実際に色味が分かりにくい場合が多い。 カラーをクリックすると別のページに遷移せず、その他の色味も分かるような仕組みになっている。 使われている技術 HTML CSS JavaScript カーソルをhoverした際に色が変わる。
